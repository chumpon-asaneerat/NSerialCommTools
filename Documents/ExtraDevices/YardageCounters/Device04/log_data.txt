========================================
LOG FORMAT NOTES
========================================

Format: HEX_BYTES_SPACE_SEPARATED   ASCII_TEXT_REPRESENTATION

- Left column: Raw bytes in hexadecimal (space-separated)
- Right column: ASCII text representation (non-printable shown as <0xHH>)
- Each line represents one serial transmission (terminated by CR/LF/CRLF)

Special Bytes:
- 0x0D = CR (Carriage Return)
- 0x0A = LF (Line Feed)
- 0x09 = TAB
- 0x20 = SPACE
